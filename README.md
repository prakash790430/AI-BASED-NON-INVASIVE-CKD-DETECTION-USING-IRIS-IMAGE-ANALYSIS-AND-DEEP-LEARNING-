# AI-BASED-NON-INVASIVE-CKD-DETECTION-USING-IRIS-IMAGE-ANALYSIS-AND-DEEP-LEARNING-
Create a research prototype system that analyzes iris images using computer vision and deep learning to simulate detection of kidney-related abnormalities.

## 📌 Project Overview
This project is a research-based web application that analyzes iris images using Artificial Intelligence and Deep Learning to simulate the detection of Chronic Kidney Disease (CKD).

> ⚠️ Disclaimer: This system is developed for academic purposes only and should not be used for real medical diagnosis.

---

## 🚀 Features
- User Authentication (Login/Register)
- Upload Iris Image
- Iris Segmentation using OpenCV
- CNN-based Classification (Healthy / Simulated CKD)
- Probability Graph Visualization
- Prediction History (SQLite Database)
- Downloadable PDF Report

---

## 🧠 Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- AI Model: TensorFlow / Keras (CNN)
- Computer Vision: OpenCV
- Database: SQLite
- Charts: Chart.js

## 📊 Model Details
- Input Size: 128x128 grayscale images  
- Architecture: CNN (Conv2D, MaxPooling, Dense)  
- Output: Binary Classification (Healthy / Simulated CKD)  
- Accuracy: ~90–95% (on simulated dataset)
